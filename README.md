🏥 Nursing College Website

A responsive and user-friendly website designed for a Nursing College to provide information about courses, admissions, facilities, and contact details.

📌 Features
🏫 Home page with college introduction
📚 Courses offered (GNM, ANM, etc.)
📝 Admission details and contact form
🏥 Facilities section (labs, hostel, etc.)
📞 Contact page with location and phone details
💻 Fully responsive design
